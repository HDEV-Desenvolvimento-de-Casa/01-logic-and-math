# Título: Desafio 001 - O Algoritmo do Caixa Eletrônico

Contexto: Um banco precisa de um script que receba um valor de saque e informe quantas notas de cada valor serão entregues. Requisitos:

Notas disponíveis: R$ 100, R$ 50, R$ 20, R$ 10.

O programa deve priorizar as notas maiores para entregar o menor número de cédulas possível.

Restrição Técnica: O aluno não pode usar frameworks. Deve ser lógica pura.

Entrega: O código deve estar limpo (Clean Code) e as variáveis devem estar em português e ser autoexplicativas.
